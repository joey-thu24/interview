# 🎓 CS Career Copilot - 产品设计方案 V2.0

> **核心定位**：计算机专业学生的一站式「学习管理 + 项目实战 + 求职陪跑」AI 智能体平台。
> **Slogan**: 不是代替你学习，而是像"技术大牛"一样带你从 0 到 1 拿到 Offer。

## 🧩 四大核心功能模块 (Product Modules)

### 1. 🧭 智能规划模块 (The Compass) —— 解决【怎么学】
*不仅仅是一个静态的 Markdown 列表，而是一个可交互的图谱。*

*   **功能特性**:
    *   **目标对齐**: 用户输入 "我想去字节做后端"，系统生成 "字节后端技能雷达图"。
    *   **动态调整**: 每天根据你的学习情况（如：今天没学完），动态调整后续计划（Supervisor Agent 升级版）。
    *   **知识胶囊**: 针对每个知识点，AI 自动聚合 "Blog + 视频 + 核心笔记 + 面试高频题"。
*   **交互形态**: 类似游戏中 "技能树" 的点亮机制。

### 2. 🏗️ 沉浸式项目工坊 (The Workshop) —— 解决【做项目】
*这是本产品的杀手级功能 (Killer Feature)。拒绝 "抄代码"，注重 "写代码"。*

*   **工作流**:
    1.  **选题**: AI 推荐高质量项目（如：手写一个 Mini-Docker，高并发秒杀系统，RPC 框架）。
    2.  **角色扮演**: **AI 扮演 Tech Lead，用户扮演 实习生**。
    3.  **需求拆解**: AI 不会直接给代码，而是给 **PRD (需求文档)** 和 **Jira 任务卡**。
        *   *任务 1*: "请设计数据库 Schema，需满足 3NF。"
        *   *任务 2*: "请实现 User 模块的接口，要求覆盖率 90%。"
    4.  **Code Review**: 用户提交代码，**AI 进行 Review**，指出命名不规范、潜在 Bug、安全性问题，直到合格才允许进入下一阶段。
*   **价值**: 真正模拟大厂开发流程，简历上可以说 "在 AI 指导下按工业级标准开发"。

### 3. 🧠 个人知识库 (Second Brain) —— 解决【记不住/复习】
*不需要用户手动整理复杂的笔记，AI 帮你构建第二大脑。*

*   **功能特性**:
    *   **自动归纳**: 学习过程中产生的对话、代码、错题，自动整理成 Notion 风格的笔记。
    *   **RAG 问答**: 用户可以随时问 "我上周学的 Redis 锁那个知识点，当时哪里错了？"
    *   **闪念胶囊**: 随时记录灵感，AI 自动打标签归类。

### 4. ⚔️ 求职演练场 (The Arena) —— 解决【怎么面】
*基于目前的版本进行深挖。*

*   **简历精修 (Resume Polish)**:
    *   上传 PDF，AI 对照 JD 进行红线扫描。
    *   **STAR 法则重写**: 将用户口语化的项目描述，改写为 "使用了 xxx 技术，提升了 xxx% 性能" 的专业描述。
*   **全真模拟**:
    *   **八股文突击**: 快速过知识点。
    *   **算法手撕**: 提供思路引导，而不是直接给答案。
    *   **情景面试**: 模拟 HR 压力面、技术总监架构面。

## 🛠️ 技术可行性与架构 (Architecture)

### 核心 Agent 设计
1.  **User Proxy**: 维护用户画像（当前水平、目标、性格）。
2.  **Curriculum Designer (规划师)**: 拥有全网主流技术栈的拓扑结构知识，负责生成 DAG (有向无环图) 学习路线。
3.  **Tech Lead (技术组长)**: 最复杂的 Agent。拥有 "代码审美" 和 "工程化标准"，负责出题和 CR 代码。
4.  **Interviewer (面试官)**: 现有的 Interviewer 模块升级。
5.  **Scribe (记录员)**: 负责后台静默整理笔记，写入 Vector DB。

### 数据流
`用户输入` -> `Supervisor 判断意图` -> `分发给对应 Agent` -> `结果存入 SQLite/ChromaDB` -> `UI 渲染`
